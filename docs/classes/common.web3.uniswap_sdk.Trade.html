Class Trade

Represents a trade executed against a list of pairs. Does not account for slippage, i.e. trades that front run this trade and move the price.

Static bestTradeExactIn

  • bestTradeExactIn(pairs: Pair[], currencyAmountIn: CurrencyAmount, currencyOut: Currency, __namedParameters?: BestTradeOptions, currentPairs?: Pair[], originalAmountIn?: CurrencyAmount, bestTrades?: Trade[]): Trade[]
    • Defined in node_modules/@uniswap/sdk/dist/entities/trade.d.ts:88

    Given a list of pairs, and a fixed amount in, returns the top maxNumResults trades that go from an input token amount to an output token, making at most maxHops hops. Note this does not consider aggregation, as routes are linear. It's possible a better route exists by splitting the amount in among multiple routes.

    Parameters

    • pairs: Pair[]

      the pairs to consider in finding the best trade

    • currencyAmountIn: CurrencyAmount

      exact amount of input currency to spend

    • currencyOut: Currency

      the desired currency out

    • Optional __namedParameters: BestTradeOptions
    • Optional currentPairs: Pair[]

      used in recursion; the current list of pairs

    • Optional originalAmountIn: CurrencyAmount

      used in recursion; the original value of the currencyAmountIn parameter

    • Optional bestTrades: Trade[]

      used in recursion; the current list of best trades

    Returns Trade[]
