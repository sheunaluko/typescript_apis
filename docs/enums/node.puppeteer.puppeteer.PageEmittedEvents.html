Console

Console = "console"
  • Defined in node_modules/puppeteer/lib/types.d.ts:5876

Emitted when JavaScript within the page calls one of console API methods, e.g. console.log or console.dir. Also emitted if the page throws an error or a warning.

remarks

A console event provides a ConsoleMessage representing the console message that was logged.

example

An example of handling console event:

page.on('console', msg => {
  for (let i = 0; i < msg.args().length; ++i)
   console.log(`${i}: ${msg.args()[i]}`);
 });
 page.evaluate(() => console.log('hello', 5, {foo: 'bar'}));

Metrics

Metrics = "metrics"
  • Defined in node_modules/puppeteer/lib/types.d.ts:5914

Emitted when the JavaScript code makes a call to console.timeStamp. For the list of metrics see page.metrics.

remarks

Contains an object with two properties:

  • title: the title passed to console.timeStamp
  • metrics: objec containing metrics as key/value pairs. The values will be numbers.

Popup

Popup = "popup"
  • Defined in node_modules/puppeteer/lib/types.d.ts:5941

Emitted when the page opens a new tab or window.

Contains a Page corresponding to the popup window.

example
const [popup] = await Promise.all([
  new Promise(resolve => page.once('popup', resolve)),
  page.click('a[target=_blank]'),
]);

const [popup] = await Promise.all([
  new Promise(resolve => page.once('popup', resolve)),
  page.evaluate(() => window.open('https://example.com')),
]);

RequestFailed

RequestFailed = "requestfailed"
  • Defined in node_modules/puppeteer/lib/types.d.ts:5969

Emitted when a request fails, for example by timing out.

Contains a HTTPRequest.

remarks

NOTE: HTTP Error responses, such as 404 or 503, are still successful responses from HTTP standpoint, so request will complete with requestfinished event and not with requestfailed.
